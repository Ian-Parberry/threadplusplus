Detailed Description

The base task descriptor describes a base task, including a task identifier, a thread identifier which is to be set by the thread that performs this task, and a function to perform it. You should derive your task desciptor from this class. Your task descriptor should implement a constructor for any task-related initialization and it should override function Perform() with the code to perform your task.
